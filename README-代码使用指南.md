以下是进行模拟的全流程：
1.模型确认
首先需要明确互作的蛋白的PDB结构，可在uniprot/RCSB PDB上寻找并确认；
2.空间结合位置设定
使用蛋白对接模拟以确认空间上较优的对接方式，我使用了pydock和haddock分别对接并交叉验证，根据rmsd选择了最优的模型。
pydock只需要提供配体和受体的.pdb文件，haddock需要填写活性位点，可使用consurf进行预测并填写。
3.模拟参数文件准备
将最终确认的对接模型上传至charmmgui-solution bulider（使用教育邮箱注册即可），按需设置每一步的参数，最终得到输出的.tgz压缩包。
4.代码运行
在colab中运行代码，请将代码分散到多个单元格运行以最大程度避免运行失败带来的损失。每一个代码文件中我都提供了可视化模拟情况的代码，以便根据情况延长或缩短模拟时间。
请善用断点接续功能，colab对非会员有8h/day的使用限制，日常使用请及时保存。
在最终的生产模拟过程中，由于模拟时间较长，如网络不稳定可能导致续跑失败，如果你和我一样将文件挂载在google drive中运行，可以将模拟所用的文件回退到相同的历史版本，然后继续模拟，因此，请定时保存并再次续跑。
